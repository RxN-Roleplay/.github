# Om oss:
Hej! Vi kallar oss RxN och driver en aktiv svensk rollspelsserver med stort fokus på fri, engagerande och rolig RP.<br>Vårt mål är att RxN ständigt ska utvecklas och bli så avancerad och välgenomtänkt som möjligt, där kvalitet och spelupplevelse alltid står i centrum.
